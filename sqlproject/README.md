# Ecommerce Sales Analysis

## Business Problem

E-commerce companies generate large amounts of sales data from different products, categories, and regions. However, without proper analysis it is difficult for businesses to understand which products perform well, which regions generate the highest sales, and which categories produce the most profit.

This project analyzes e-commerce sales data using SQL and Power BI to identify important business insights such as top-performing categories, profitable products, and regional sales performance.

## Business Objective

The objective of this project is to analyze e-commerce sales data and provide meaningful insights to support business decision making.

The analysis focuses on:

• Identifying top-selling product categories
• Finding the most profitable products
• Analyzing sales performance by region
• Calculating total sales, profit, and quantity sold
• Visualizing key insights using a Power BI dashboard

## Dataset

The dataset used in this project contains e-commerce sales data including product name, category, region, sales amount, profit, and quantity.

## Tools Used

• MySQL
• SQL
• Power BI
• GitHub

## SQL Analysis

Multiple SQL queries were written to analyze the dataset. These queries include:

• Total sales calculation
• Category wise sales analysis
• Region wise sales analysis
• Product wise profit analysis
• Top selling products identification
• Average sales and profit calculations
